## Regions/availability zones

As regiões são áreas geográficas onde a AWS mantém seus datacenters.
Zonas de Disponibilidade (Availability Zones)
Cada região possui uma ou mais zonas de disponibilidade

## O que é o Amazon EC2 e armazenamento em nuvem EBS E S3

O Amazon Elastic Compute Cloud (EC2) é um serviço da AWS que permite criar e executar servidores virtuais na nuvem.
O Armazenaento em nuvem EBS e uma extensão de armazenamento mesmo conceito de hd externo e S3 eo armazenamento de objetos altamente durável e escalável.

## Importância do Controle de Custos na AWS

O controle de custos é um dos aspectos mais importantes da utilização da AWS, pois os serviços são cobrados com base no consumo. Sem um monitoramento adequado, recursos podem permanecer ativos desnecessariamente, gerando cobranças inesperadas.
